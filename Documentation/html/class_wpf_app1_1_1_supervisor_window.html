Detailed Description

SupervisorWindow.

Logika interakcji dla okna SupervisorWindow.xaml. Okno stanowi panel administracyjny (Przełożonego), umożliwiający całościowy wgląd w bazę klientów i pracowników oraz dodawanie nowego personelu do systemu.
